Acme Email - Message Processor
=============

A Simple AngularJS Demo for WeConvene.


Built on Ubuntu with gVim, Compass, Sass and Testacular


To Run Unit Tests:
------------------

###Linux: 

"sh scripts/test.sh"


###Windows (untested): 

"cd scripts"
"test"


To Run End to End Tests:
------------------

###Linux: 

"sh scripts/e2e-text.sh"


###Windows (untested): 

"cd scripts"
"e2e-test"
